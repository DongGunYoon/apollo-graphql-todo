# How To Use

## Project setup
```
npm install
or
yarn
```

### Compiles and hot-reloads for development
```
npm run start
or
yarn run start
```

### Compiles and minifies for production
```
npm run build
or
yarn run build
```

### Lints and fixes files
```
npm run lint
or
yarn run lint
```

### URL Address
* GrpahQL Playground: <http://localhost:3000/graphql>
* 클라이언트 주소: <http://localhost:8080/login>
